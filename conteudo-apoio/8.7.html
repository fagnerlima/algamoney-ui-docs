Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

Angular CLI

Visualizando a versão

Na aula, para exibir a versão instalada do Angular CLI, o seguinte comando é demonstrado:

[code lang="shell"] ng -v [/code]

No Angular CLI 10 utilizamos o seguinte comando:

[code lang="shell"] ng version [/code]

Criação interativa do projeto

Quando criamos um novo projeto com o Angular CLI 10, são apresentadas algumas perguntas com opções de configuração.

Angular Routing

Será exibida uma mensagem perguntando se desejamos adicionar Angular Routing no nosso projeto. Para o nosso caso, como adicionaremos mais a frente essas funcionalidades, podemos digitar N e continuar com a criação do projeto

CSS

Também será perguntado qual formato para folhas de estilo iremos usar em nosso projeto. Para o nosso caso utilizaremos CSS tradicional.

1. Notas de atualização

O processo de atualização e instalação é o mesmo que mostrado na aula, com algumas sutis diferenças:

1.1. Visualizando a versão

Para exibir a versão instala do Angular CLI, na aula é demonstrado da seguinte forma:

[code lang="shell"] ng -v [/code]

Para essa versão, utilizamos o seguinte comando:

[code lang="shell"] ng version [/code]

1.2. Criação interativa do projeto

Ao digitar o comando para a criação de um novo projeto, para a versão atual, o CLI irá nos fazer algumas perguntar

Angular Routing

Será exibido uma mensagem perguntando se desejamos adicionar Angular Routing no nosso projeto. Para o nosso caso, como adicionaremos mais a frente essas funcionalidades, podemos digitar N e continuar com a criação do projeto

CSS

Também será perguntado qual formato para folhas de estilo iremos usar em nosso projeto. Podemos escolher entre algumas opções, mas para o nosso caso iremos usar CSS tradicional.
