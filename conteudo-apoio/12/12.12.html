Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

Bootstrap

Algumas classes de validação do Bootstrap foram removidas e outras alteradas. As classes form-control-label e form-control-feedback foram removidas e a validação para o label agora precisa ser customizada.

A classe has-danger também foi removida, porém temos agora as classes is-valid e is-invalid, sendo is-invalid o que procuramos para substituir a classe has-danger.

Para o feedback também existem duas outras classes: valid-feedback e invalid-feedback. Utilizaremos a invalid-feedback.

[code lang="html"] <form autocomplete="off" #usuarioForm="ngForm" (ngSubmit)="salvar(usuarioForm)"> <div class="form-group"> <label>Nome</label> <input type="text" class="form-control" name="primeiroNome" [(ngModel)]="cliente.nome" required minlength="10" #nome="ngModel" [class.is-invalid]="nome.invalid && nome.touched"> <div *ngIf="nome.hasError('required') && nome.touched" class="invalid-feedback"> Informe um nome </div> <div *ngIf="nome.hasError('minlength') && nome.touched" class="invalid-feedback"> O nome deve ser completo </div> </div> </form> [/code]

Como visto acima, precisamos apenas informar as classes is-invalid para os componentes form-control e invalid-feedback para os componentes de feedback.
