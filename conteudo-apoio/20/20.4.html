Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

Configuração do package.json

A única diferença que teremos nessa aula, é o nome da propriedade que o heroku utilizará para fazer o build.

Na aula utilizamos a propriedade "postinstall", dessa vez precisamos apenas renomear para "heroku-postbuild"

[code lang="javascript"] "scripts": { //... "heroku-postbuild": "ng build --prod" }, [/code]

Configuração do angular.json

O @angular/cli 10 deposita os arquivos compilados numa pasta com o nome do projeto dentro da pasta "dist". A seguinte configuração pode ser vista no arquivo "angular.json":

[code lang="json"] "outputPath": "dist/algamoney-ui" [/code]

Para que não precisemos alterar a configuração do arquivo "server.js", devemos alterar a seguinte configuração do "outputPath" no arquivo "angular.json":

[code lang="json"] "outputPath": "dist" [/code]
