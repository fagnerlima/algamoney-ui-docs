Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

1. Utilizando environment de produção

Se tentarmos utilizar o comando mostrado na aula iremos nos deparar com um erro:

[code lang="bash"] $ ng build --environment=prod [/code] Unknown option: '--environment'

Isso se dá pelo fato dessa opção ter sido alterada em versões mais recentes.

Para resolvermos este problema devemos ajustar o comando, utilizando da seguinte forma:

[code lang="bash"] ng build --configuration=production [/code]
