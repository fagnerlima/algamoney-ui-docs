Sintaxe do Lazy Loading

A sintaxe do lazy loading com Angular 10 é um pouco diferente:

[code lang="typescript"] { path: 'lancamentos', loadChildren: () => import('app/lancamentos/lancamentos.module').then(m => m.LancamentosModule) }, { path: 'pessoas', loadChildren: () => import('app/pessoas/pessoas.module').then(m => m.PessoasModule) }, [/code]
