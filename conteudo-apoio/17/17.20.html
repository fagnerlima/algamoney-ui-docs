Requisição

Podemos utilizar tipagem no método post para que possamos obter a resposta diretamente como Lancamento:

[code lang="javascript"] adicionar(lancamento: Lancamento): Promise { const headers = new HttpHeaders() .append('Authorization', 'Basic YWRtaW5AYWxnYW1vbmV5LmNvbTphZG1pbg==') .append('Content-Type', 'application/json'); return this.http.post(this.lancamentosUrl, lancamento, { headers }) .toPromise(); } [/code]

Também, como visto acima, podemos enviar o objeto lancamento sem precisar transformá-lo em string.

Por fim, não precisamos também converter a resposta utilizando o metodo "json()"
