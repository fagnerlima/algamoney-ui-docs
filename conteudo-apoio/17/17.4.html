Parâmetros

Como vimos na aula anterior, HttpParams é um objeto imutável, por isso precisamos reatribuir seu valor quando usamos o método set:

[code lang="javascript"] if (filtro.dataVencimentoInicio) { params = params.set('dataVencimentoDe', moment(filtro.dataVencimentoInicio).format('YYYY-MM-DD')); } if (filtro.dataVencimentoFim) { params = params.set('dataVencimentoAte', moment(filtro.dataVencimentoFim).format('YYYY-MM-DD')); } [/code]

Mensagem na tabela sem registros

Na aula é demonstrada a utilização do parametro emptyMessage para customizar a mensagem que aparece em nossa tabela quando não houverem dados para exibir.

Esta propriedade não existe no componente Table, porém a forma para customizarmos é bem simples e parecida:

[code lang="html"] <ng-template pTemplate="emptymessage"> Nenhum lançamento encontrado </ng-template> [/code]

Observe que foi definido o valor 6 para o colspan, isso porque existem 6 colunas na nossa tabela de lançamentos.
