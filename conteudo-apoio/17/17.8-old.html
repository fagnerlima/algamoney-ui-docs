Nota para atualização de dependências

Este documento irá te auxiliar a fazer esta aula com a versão mais recente do Angular e suas dependências. Atualmente a versão mais recente do Angular é a 8.

Observação: Este documento é apenas para os que querem usar as versões mais recentes das dependências contidas nesta aula. Ao longo do curso iremos atualizar as dependências, mas caso queira atualizar agora, use este documento como guia.

1. Mudança no decorador @ViewChild

Na versão 8 do Angular, foi alterado a forma como o decorador @ViewChild é avaliado.

Agora é necessário especificar a propriedade static, que irá dizer ao Angular em que momento as mudanças deverão ser detectadas.

No nosso caso, adicionaremos a opção static = true:

[code lang="javascript"] @ViewChild('tabela', {static: true}) grid: Table; [/code]

2. Reiniciando o grid

Por utilizarmos o componente Table nesta versão, apenas indicar o valor da propriedade first, não vai gerar o resultado esperado.

Este componente, por outro lado possui um método chamado "reset". Iremos usar este método para reiniciar o estado da nossa tabela.

[code lang="javascript"] excluir(lancamento: any) { this.lancamentoService.excluir(lancamento.codigo) .then(() => { this.grid.reset(); }); } [/code]
