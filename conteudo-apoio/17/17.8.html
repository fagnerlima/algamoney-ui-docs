Reiniciando a tabela

Por utilizarmos o componente Table nesta versão, apenas indicar o valor da propriedade first não vai gerar o resultado esperado. Este componente, por outro lado, possui um método chamado "reset". Iremos usar este método para reiniciar o estado da nossa tabela.

[code lang="javascript"] excluir(lancamento: any) { this.lancamentoService.excluir(lancamento.codigo) .then(() => { this.grid.reset(); }); } [/code]
