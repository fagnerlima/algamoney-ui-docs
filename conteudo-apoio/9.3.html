Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

Criação de componentes

Os passos para se criar um componente, como demonstrado na aula, são exatamente os mesmos para esta versão. A única ressalva para esta aula fica por conta do parâmetro --spec:

[code lang="shell"] --spec=false [/code]

Caso execute o comando utilizando esse parâmetro, irá se deparar com uma mensagem dizendo que a opção é desconhecida. Isso acontece pois ele foi substituído nas últimas versões. O seu substituto é o --skip-tests, que pode ser utilizado nas seguintes formas:

[code lang="shell"] --skip-tests=true [/code] [code lang="shell"] --skipTests=true [/code]

Ou em forma simplificada:

[code lang="shell"] --skip-tests [/code] [code lang="shell"] --skipTests [/code]

1. Dicas de atualização

Os passos para se criar um componente, como demonstrado na aula, são exatamente os mesmos para essa versão.

A única ressalva para esta aula seria o parametro:

[code lang="shell"] --spec=false [/code]

Caso rode o comando utilizando esse parâmetro, irá se deparar com uma mensagem dizendo que ele está depreciado.

Isso significa que ele não é mais utilizado e foi substituído.

O seu substituto é o skip-tests, e pode ser utilizado nas seguintes formas:

[code lang="shell"] --skip-tests=true [/code] [code lang="shell"] --skipTests=true [/code] Ou em forma simplificada: [code lang="shell"] --skip-tests [/code] [code lang="shell"] --skipTests [/code]
