Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

PrimeNG 10

Tabela responsiva

Para tornar nossa tabela responsiva no PrimeNG 10, primeiramente utilizamos o mesmo atributo mostrado na aula: responsive. Além disso, precisamos adicionar a informação de cabeçalho no template body, pois ele que será exibido nos dispositivos de menor resolução.

O cabeçaho no corpo da tabela deve ser adicionado com a classe CSS p-column-title da seguinte forma:

[code lang="html"] Pessoa {{ lancamento.pessoa }} [/code]

Segue o código completo da tabela responsiva:

[code lang="html"] Pessoa Descrição Vencimento Pagamento Valor Pessoa {{ lancamento.pessoa }} Descrição {{ lancamento.descricao }} Vencimento {{ lancamento.dataVencimento }} Pagamento {{ lancamento.dataPagamento }} Valor {{ lancamento.valor }} [/code]

Além disso, precisamos ajustar nossas regras CSS para tabelas para que elas não sejam aplicadas em dispositivos de menor resolução, pois não precisamos, por exemplo, centralizar informações da tabela, pois todas elas serão exibidas com alinhamento à esquerda. Para ajustar o CSS faremos uso de media query com min-width: 40em, que é o valor de resolução que o PrimeNG utiliza para alterar o layout da tabela para dispositivos de menor resolução.

Segue o código completo das regras CSS da tabela:

[code lang="css"] @media (min-width: 40em) { .col-valor-header { width: 120px; } .col-valor { text-align: right !important; } .col-data-header { width: 120px; } .col-data { text-align: center !important; } .col-acoes-header { width: 115px; } .col-acoes { width: 115px; text-align: center !important; } .col-acoes .p-button { margin: 1px; } } [/code]
