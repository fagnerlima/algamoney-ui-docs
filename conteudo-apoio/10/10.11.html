Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

PrimeNG 10

Ícones

Como dito anteriormente, os componentes de ícones que estamos usando agora são do PrimeIcons. Por esta razão, devemos atualizar o ícone do nosso menu:

[code lang="html"] <i class="pi pi-bars"></i> [/code]

CSS

Um pequeno ajuste no CSS é necessário para centralizar nosso ícone de menu na navbar:

[code lang="css"] .navbar { padding: 20px 0 10px; } [/code]
