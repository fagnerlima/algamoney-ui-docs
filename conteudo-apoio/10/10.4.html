Atualização para o Angular 10

Este documento irá auxiliá-lo no uso do Angular 10.

Observação: Este documento é apenas para os que querem utilizar o Angular na versão 10. Ao longo do curso atualizaremos as dependências, mas caso queira atualizá-las agora, use este documento como guia.

Importação de módulos do PrimeNG 10

Os componentes no PrimeNG 10 são configurados como módulos. Uma vez que o PrimeNG está instalado e configurado, módulos e APIs podem ser importados com o padrão de path primeng/{module}. A documentação de cada componente apresenta a sua respectiva importação.

Segue exemplo da documentação do PrimeNG 10:

[code lang="typescript"] import { AccordionModule } from 'primeng/accordion'; // accordion and accordion tab import { MenuItem } from 'primeng/api'; // api [/code]

Nesta aula as alterações que devem ser feitas no AppModule são as seguintes:

[code lang="typescript"] import { InputTextModule } from 'primeng/inputtext'; import { ButtonModule } from 'primeng/button'; [/code]

Classes CSS do PrimeNG 10 e do PrimeFlex

Algumas classes CSS do PrimeNG foram substituídas na versão 10. Anteriormente, por padrão, as classes CSS do PrimeNG tinham o prefixo ui-, passando agora para o prefixo p-. Além do prefixo são necessárias apenas algumas alterações em casos específicos como o da classe ui-g que deve ser substituída pela classe p-grid.

Segue um resumo de algumas alterações:

Classe removida Nova classe Observação
ui-g p-grid
ui-g-x p-col-x x = 1~12
ui-md-x p-md-x x = 1~12
ui-lg-x p-lg-x x = 1~12
ui-fluid p-fluid
ui-button p-button
ui-button-* p-button-*
ui-dropdown p-dropdown
ui-message p-message
ui-message-* p-messages-*
