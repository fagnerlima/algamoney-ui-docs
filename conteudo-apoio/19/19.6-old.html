1. Tratamento da resposta

Na aula, vimos que para obter a propriedade erro da resposta, vinda da API, primeiro era necessário converter o objeto response para json, utilizando o método "json()"

Como com o HttpClient, não usamos o método json, precisamos obter o erro, através da propreidade "error". Ajustando o código da aula, teríamos:

[code lang="java"] .catch(response => { const responseError = response.error; if (response.status === 400) { if (responseError.error === 'invalid_grant') { return Promise.reject('Usuário ou senha inválida'); } } return Promise.reject(response); }); [/code]
