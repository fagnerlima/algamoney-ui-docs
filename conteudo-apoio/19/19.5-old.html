Nota para atualização de dependências

Este documento irá te auxiliar a fazer esta aula com a versão mais recente do Angular e suas dependências. Atualmente a versão mais recente do Angular é a 8.

Observação: Este documento é apenas para os que querem usar as versões mais recentes das dependências contidas nesta aula. Ao longo do curso iremos atualizar as dependências, mas caso queira atualizar agora, use este documento como guia.

1. Dependência Angular2-JWT

A dependência que precisaremos instalar em nosso projeto agora, mudou, pois a versão utilizada na aula não é compatível com o módulo HttpClient.

Segue o comando de instalação:

[code] npm install @auth0/angular-jwt [/code]

2. JwtHelperService

Precisaremos também declarar no seção "providers" de nosso módulo (seguranca.module.ts) o seguinte serviço:

[code lang="javascript"] providers: [JwtHelperService] [/code]

3. JwtModule

Precisamos também realizar uma configuração bem simples para que o módulo possa funcionar corretamente.

Também em nosso arquivo seguranca.module.ts, precisamos adicionar o seguinte código na seção "imports":

[code lang="javascript"] JwtModule.forRoot({ config: { tokenGetter: () => { return ''; } } }) [/code]

Pronto, agora estamos prontos para utilizar a dependência para JWT

4. Utilização

Não há diferenças na forma de utilizar essa biblioteca, comparado com o que foi ensinado na aula, as únicas diferenças são os tipos dos nossos atributos e de onde vamos importar.

Em nosso construtor, na classe AuthService, iremos receber como parametro um JwtHelperService, do pacote "@auth0/angular-jwt"

[code lang="javascript"] //... import { JwtHelperService } from '@auth0/angular-jwt'; //... constructor( private jwtHelper: JwtHelperService, //.. ) { //... } [/code]

Já a forma de decodificar, e como armazenar este token, será da mesma maneira da aula.
